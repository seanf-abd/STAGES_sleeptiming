# STAGES_test
Attempted workflow for STAGES dataset, all dirs are local atm so this will have to be updated in time
Currently there are three separate .Rmd's that should be used in the following order
1. Actigraphy_workflow
2. PSG_workflow
3. Plotting
There are multiple comments in each of the .Rmd's indicating the places where you will need to update the directory to your own relevant file/server location.
Any instance of data being required directly from the NSRR website should be signposted but please let me know if this is not the case and I'll update accordingly.

I am looking to merge this code with Ruby so it is easier to use the data directly from the NSRR website.
